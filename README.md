# Facial Expression Recognition

Webcam demo for tracking people, locating faces, and classifying facial expressions.

The current expression model is EmotiEffLib `enet_b2_8`, which predicts:

- `Anger`
- `Contempt`
- `Disgust`
- `Fear`
- `Happiness`
- `Neutral`
- `Sadness`
- `Surprise`

## Run

```powershell
python src/main.py
```

Press `q` to close the camera window.
